# Minecraft Engineering Workshop

このリポジトリは、GitHub と NEC が共同で開催した  
Minecraft ワークショップで作成されたワールドを共有するためのリポジトリです。

ワークショップ終了後も、参加者がワールドをダウンロードして続きを楽しめるように公開しています。

## 使い方

1. `.mcworld` ファイルをダウンロード  
2. ファイルをダブルクリック  
3. Minecraft にワールドがインポートされます
