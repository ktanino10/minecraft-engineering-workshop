# Minecraft Engineering Workshop

このリポジトリは、GitHub と NEC が共同で開催した  
Minecraft ワークショップで作成されたワールドを共有するためのリポジトリです。

ワークショップ終了後も、参加者がワールドをダウンロードして続きを楽しめるように公開しています。

---

## 前提条件

このワールドを利用するには、以下の準備が必要です。

- Minecraft Education が動作する PC
- Minecraft Education のインストール

Minecraft Education についてはこちらをご確認ください。

https://education.minecraft.net/

※PCやMinecraft Educationのライセンスは各自でご用意ください。

## 使い方

1. `.mcworld` ファイルをダウンロード  
2. ファイルをダブルクリック  
3. Minecraft にワールドがインポートされます
